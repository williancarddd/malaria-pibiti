# Treinamento para Redes Neurais
Aqui realizamos o treinamento para redes neurais garantindo  a mesma saida para diferentes redes.

- classes
-- data: processamento dos dados, conversão para RGB, normalização e crop.
-- model: criação do modelo, treinamento e avaliação.
metrics.py: métricas de avaliação utilizada durante todo o projeto.

- config
config.py: configuração de video, batch_size, epochs, etc.

main.py: script principal para treinamento e avaliação.
